# xale-tls

This repo is part of the [xale-db](https://github.com/axdelafuen/xale-db) project.

This is a simple PoC implementation of the SSL/TLS protocol, written in `c++`.

---

Its purpose is to evaluate whether building and maintaining a custom SSL/TLS implementation is feasible for [xale-db](https://github.com/axdelafuen/xale-db).

If this project doesn’t work out. A third-party library will be used instead.
